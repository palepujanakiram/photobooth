# Slideshow images

The consent screen uses them for the animated slideshow background. You can add fewer than 19; missing files will show a placeholder.

**Naming:** `slideshow_1` through `slideshow_19`. In this repo:

- **JPEG:** `slideshow_1.jpg` … `slideshow_10.jpg`, and `slideshow_19.jpg` (photos; smaller than full-size PNGs).
- **PNG:** `slideshow_11.png` … `slideshow_18.png`.

To use a different extension for a slot, update `kSlideshowAssetPaths` in `lib/views/widgets/animated_slideshow_background.dart`.
